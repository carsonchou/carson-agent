# 瘋狂優化自適應策略 + 選股濾網（train/test OOS）

- 全台股修分割、≥5年；依代碼末位 偶=train(883) / 奇=test(887)。
- 搜尋只在 train（隨機子樣本 400 檔，seed 固定）；最後 test 報 OOS。
- Fitness = 濾後池 獲利檔比例 + 0.3×中位Ret/MaxDD（池<40 重罰）。

## 誠實結論

- 選股濾網是大槓桿：把 OOS 獲利檔比例從 baseline 全市場 54% 拉到濾後池 **88%**（全期選股）/ **62%**（滾動可實作）。
- **這是「選股後的子集」（110 / 887 檔），不是全台股**。濾掉的是低流動性/長期陰跌/無趨勢的股——它們本來就不該套這策略。
- ⚠ 滾動可實作版（62%）明顯低於全期選股版（88%）→ 全期選股的好看有相當部分來自**未來資訊**，實作時打折。
- 定論：**沒有策略能讓全台股都賺；但『自適應策略 + 選股濾網』能把可用子集的勝率顯著拉高**。實務 = 先濾股、再對篩出來的池套自適應。
