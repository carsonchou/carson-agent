# 台股策略參數優化結果（train/test 防過擬合）

- 成本模型：`tw_real`（真實台股手續費 0.1425%×2 + 證交稅 0.3%）
- 資料門檻：≥5 年、≥1000 根
- 切分：股票代碼末位 偶數→train、奇數→test
- Train 樣本：150 檔（母體 883 檔，seed=42）
- Test 樣本：150 檔（母體 887 檔，seed=42，OOS）
- Fitness：`score = 獲利檔比例 + 0.1*(平均PF-1)`

## 最佳參數（train 上 coordinate descent）
