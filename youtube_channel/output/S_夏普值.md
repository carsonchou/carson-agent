# 🎬 夏普值在看什麼

## ⚡ HOOK（0-5 秒）

**旁白：** 同樣賺10%,這支策略卻更好...關鍵在夏普值。

**建議畫面：** Same 10% Return / Sharpe Ratio / Which One Is Better?

## 📦 主體

### 段落 1：夏普值在量什麼

**旁白：** 夏普值衡量的是，你每多承擔一單位的風險，能換回多少報酬。報酬一樣的兩支策略，波動小、回檔淺的那支，夏普值就比較高，代表它賺得更穩、更省心。

**建議畫面 / B-roll：** Sharpe Ratio = Return / Risk、Smooth Equity Curve vs Volatile Curve

### 段落 2：別只盯總報酬

**旁白：** 很多人只盯著總報酬，卻忽略了過程中的劇烈震盪。帳戶忽上忽下的煎熬，往往才是讓人提早下車、放棄策略的真正原因。

**建議畫面 / B-roll：** Drawdown Pain、Account Equity Swings、Don't Just Look At Total Return

## 🏁 結尾（OUTRO）

**旁白：** 所以下次看回測，記得看夏普值。想看完整版的觀念拆解，追蹤量化阿森。

**建議畫面：** Check Sharpe In Backtest、Follow Carson Quant
