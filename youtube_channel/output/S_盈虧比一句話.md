# 🎬 盈虧比比勝率更重要

## 📦 主體

### 段落 1：什麼是盈虧比

**旁白：** 盈虧比,就是你平均賺一筆,跟平均賠一筆的比例。很多人勝率七成、八成,聽起來超強,但每次只賺一點點、賠卻賠很大。這種小賺大賠的系統,看起來在贏,其實在慢性死亡。

**建議畫面 / B-roll：** risk reward ratio diagram, small green bar vs big red bar, slow bleeding equity curve

### 段落 2：真正決定賺賠的公式

**旁白：** 算法很簡單:長期能不能賺,看的是勝率乘以平均獲利,有沒有大於敗率乘以平均虧損。所以與其拚勝率,不如先把停損守好、讓獲利跑久一點,把盈虧比拉上來。

**建議畫面 / B-roll：** expected value formula on screen, stop loss line, let profits run arrow, calculator

## 🏁 結尾（OUTRO）
