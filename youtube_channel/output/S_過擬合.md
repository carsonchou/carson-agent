# 🎬 回測完美卻實盤虧爆

## ⚡ HOOK（0-5 秒）

**旁白：** 回測曲線完美到不真實?小心,你可能不是找到聖杯,而是掉進過擬合的陷阱。

**建議畫面：** perfect equity curve、"too good to be true" text overlay、zoom in on smooth backtest line

## 📦 主體

### 段落 1：什麼是過擬合

**旁白：** 很多人為了讓回測賺更多,把參數調了又調,結果策略只是死記了這段歷史行情。市場一變,馬上破功,實盤直接虧爆。

**建議畫面 / B-roll：** overfitting curve vs real data、parameter sliders moving、backtest vs live PnL gap、red drawdown chart

### 段落 2：怎麼避開

**旁白：** 真正能用的策略,要在沒看過的資料上也穩,這叫樣本外測試。少調幾個參數,留一段資料不碰,反而更接近真實。回測太美,通常是警訊,不是聖杯。

**建議畫面 / B-roll：** out-of-sample test、train test split diagram、walk forward analysis、simple robust strategy
