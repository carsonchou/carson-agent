## 一、播放清單規劃（依四支柱 + Shorts + 系列）

播放清單是頻道的「貨架」，讓觀眾一眼看懂你在賣什麼、能一路看下去（拉觀看時數，直接幫 YPP）。每支長片**只歸一個主支柱清單**（避免重複稀釋），但可同時放進「系列／主題清單」。

| # | 播放清單名稱 | 對應支柱 | 收錄範圍 | 排序原則 |
|---|---|---|---|---|
| P1 | **策略拆解 ｜ Strategy Decode** | 策略拆解 | 拆解單一策略邏輯（網格、馬丁、定投、Triple SuperTrend）：為什麼這樣設、什麼行情失效 | 由淺入深，新手友善的放最前 |

## 三、片尾資訊卡（End Screen）與置頂留言模板

### 3-1 片尾資訊卡（End Screen，最後 5–20 秒）

YouTube 片尾畫面建議放 3 個元素，固定版位養成觀眾肌肉記憶：

| 版位 | 元素 | 內容/規則 |
|---|---|---|
| 左 | **下一支影片**（最佳引導） | 選「最相關的下一步」——例如本片是「網格能不能賺」就接「網格關鍵參數」。優先同系列/同路線圖 |
| 右上 | **訂閱按鈕**（頻道頭像） | 固定位置，旁白同步喊「訂閱＋小鈴鐺」 |
| 右下 | **播放清單 或 觀眾自選**（YouTube 推薦） | 導向對應支柱播放清單（P1–P7），延長 session |

> 製作規範：片尾卡背景沿用統一品牌色＋字體模板，預留最後 15–20 秒乾淨畫面放卡片，旁白在此段做 CTA（見各腳本 OUTRO）。

### 3-2 卡片（Cards，影片中段彈出）

- 在提到「我之前拆過 OO」「下一支會講 XX」時，於該秒數掛**對應影片卡**（如本片講原理、提到參數時掛「關鍵參數」那支）。
- 每支長片建議掛 2–4 張卡，分散在中後段，避免開頭就跳出干擾完播。

### 3-3 置頂留言模板（每片上架後立即置頂）

**通用模板（長片）：**

```

🤖 想自己跑派網機器人的，註冊連結放資訊欄，用我的推薦碼有返佣（對你沒額外費用，要不要用你自己決定）。

⚠️ 以上為教學與資訊分享，不是投資建議。自動交易與加密貨幣高風險，過去績效不代表未來，請用閒置資金、自行判斷並為決策負責。

```

## 四、品牌一致性檢查表（每支上架前逐項過）

> 頻道整理部門在影片進入上架前做 metadata 與門面一致性把關；任一項不過 → 退回對應部門修正。

### 4-1 視覺與識別
- [ ] 縮圖採用統一品牌色 + 同一字體模板，標題字 ≤ 6 字重點，可讀於手機小圖
- [ ] 片頭/片尾沿用同一模板與品牌色
- [ ] 字幕/重點字卡字型、位置、配色一致
- [ ] 頻道頭像、橫幅、浮水印（影片右下訂閱浮水印）已設定且一致

### 4-2 命名與 metadata
- [ ] 標題格式一致（主標題清楚 + 必要關鍵字，不誇大不農場標）
- [ ] 描述含：一句話主旨 → 三重點 → 頻道定位句 → **風險聲明** → 聯盟連結說明 → Hashtags
- [ ] Hashtags 命中核心關鍵字（#量化交易 #Pionex #派網 #網格交易 #自動交易 …）
- [ ] 影片已歸入「正確且唯一」的主支柱播放清單（P1–P4），必要時加掛系列清單（P6/P7）/Shorts（P5）
- [ ] 章節（Chapters）時間戳已設（長片 ≥ 8 分鐘者）

### 4-3 內容與調性
- [ ] 旁白為冷靜理性顧問口吻（非叫賣、非情緒），固定 voice profile（Daniel／eleven_multilingual_v2）
- [ ] Hook 在前 5–15 秒拋痛點或反直覺結論
- [ ] 結構完整：Hook → 正文（分段一段一重點）→ CTA

### 4-4 誠信鐵則（最高優先，違反即退件）
- [ ] **無編造個人實測損益**、**無保證收益**、**無喊單/報明牌**
- [ ] 風險聲明常駐於描述與（必要時）片中字卡
- [ ] 聯盟連結為**軟推**並標明「推薦連結／返佣／對你無額外費用」
- [ ] 引用數據/外部資訊有標來源；以教學與觀念為主

### 4-5 上架前最後檢查
- [ ] 片尾資訊卡 3 元素到位、卡片（Cards）已掛
- [ ] 置頂留言已準備（上架後立即發布並置頂）
- [ ] 縮圖與標題不互相矛盾、不做標題殺人（clickbait 但內容不符）
