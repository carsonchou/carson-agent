# 宣傳與社群企劃

> 頻道：量化阿森｜Carson Quant
> 定位：量化／自動交易教學（網格、定投、派網 Pionex、回測、風控）
> 受眾：對自動交易有興趣的新手到進階散戶，繁體中文使用者

---

## 誠信鐵則（全部文案皆須遵守）

1. **不編造個人實測損益**：所有數字若非真實回測／公開資料，一律標「示意」「假設」「教學用」，不冒充本人實盤戰績。
2. **不保證收益、不喊單**：禁用「穩賺」「保證」「翻倍」「跟單必勝」「報明牌」等字眼；談策略只談邏輯與機率，不提供買賣指令。
3. **聯盟連結軟推＋風險聲明**：Pionex／交易所連結一律標註「合作連結」，並附「投資有風險、過往績效不代表未來」聲明。
4. **教學／觀念為主、引用標來源**：引用數據、研究、他人觀點時標明出處；定位是知識分享而非投顧。
5. **不買量、不洗讚、不洗留言、不洗追蹤、不洗版**：跨平台僅做合規分享，尊重各社群規則。

> 統一風險聲明（短版，可貼於各則結尾）：
> ⚠️ 本內容為教學與觀念分享，非投資建議。量化／自動交易仍有虧損與爆倉風險，過往回測不代表未來績效，請自行評估並對自己的資金負責。

## 二、跨平台導流文案 ×4（合規、不買量、不 spam）

> 共同原則：每平台用該平台語感改寫，不一字不改硬貼；附自身價值（觀念點）而非純丟連結；遵守各站張貼規則，不洗版、不私訊轟炸。

### Threads（短、口語、拋問句）

```
網格機器人不是「開了就躺賺」。

我自己整理新手最常死的三點：
① 區間設太窄 ② 網格數亂抓 ③ 沒想好怎麼離場

第三點最常被忽略，也最痛。
你開過網格嗎？最後悔哪個設定？👇

（完整拆解放頻道，主頁連結。教學分享，非投資建議，自動交易有風險。）
```

### Instagram（圖文／Reels 說明欄，重視收藏與標籤）

```
📌 開網格前先存這張：新手三大致命設定

1️⃣ 上下限設太窄 → 行情一走就出場套住
2️⃣ 網格數量亂抓 → 手續費吃掉價差
3️⃣ 沒設離場條件 → 趨勢盤被慢慢套牢

（圖卡為教學示意，非實盤損益）

想看每一點怎麼修正 → 主頁連結到完整影片 🔗
覺得有用先收藏 🔖 之後設定前拿出來對一次。

⚠️ 教學分享非投資建議，自動交易有虧損風險。
#量化交易 #網格交易 #Pionex #派網 #自動交易 #加密貨幣 #定投 #理財知識
```

### Facebook（社團／粉專，敘事稍長、引導討論）

```
最近常被問：「網格機器人到底穩不穩？」

老實說，網格本身沒有穩不穩，是「人怎麼設」決定的。
我把最常見的三個翻車原因整理成一支教學影片：
區間、網格數、離場條件——錯一個，震盪盤都還好，遇到趨勢盤就會被慢慢套住。

影片裡用示意圖（非實盤損益）說明每一種狀況怎麼發生、怎麼調整。
分享給也在研究自動交易的朋友，少踩一個坑是一個。

▶ 影片連結：（連結）

※ 內容為教學與觀念分享，非投資建議；自動交易有虧損與爆倉風險，請自行評估。
（影片中交易所連結為合作連結）
```

> Facebook 注意：貼到社團前先確認該社團規則是否允許帶連結／自媒體分享；不允許者只分享觀念、不放連結，避免被視為廣告。

### Dcard（理財／投資版，重原創討論、忌業配感）

```
標題：研究自動交易半年，整理一下「網格新手最常踩的三個雷」

最近在自學量化跟自動交易，網格大概是最多人入門的。
但我發現很多人（包含一開始的我）會死在很基本的設定上，分享一下心得，純交流：

1. 區間（上下限）設太窄
   想說範圍小成交多，結果行情一走就出界，剩半套不套的部位卡著。

2. 網格數量亂抓
   格子切太密，價差還沒蓋過手續費；切太疏又抓不到震盪。

3. 沒想清楚怎麼離場
   震盪盤很爽，遇到單邊趨勢沒有離場條件就被慢慢套。

這些都是觀念，沒有要報明牌也沒有要喊單，數字部分我都用回測示意在理解，
不是實盤戰績（這點想先講清楚，避免誤會）。

有在跑網格的卡友，你們最後悔哪個設定？或有什麼我沒提到的雷？
想聽聽不同經驗～

（補充：投資／自動交易都有風險，回測不代表未來，這篇只是分享學習心得不是建議。）
```

> Dcard 注意：避免在內文直接貼商業連結或頻道導流，容易被檢舉廣告／自我宣傳；以原創討論為主，連結需求放個人簡介或待網友主動詢問。

## 三、互動企劃（問答 + 投票題庫）

### A. 問答題（留言區引導，鼓勵分享經驗）

| 編號 | 問題 | 目的 |
|---|---|---|
| Q1 | 你開的第一個機器人是哪種？網格、定投還是合約？結果如何（不用報金額）？ | 拉自我揭露、建立社群感 |
| Q2 | 你設網格時，最常憑感覺亂抓的是哪個參數？ | 連結 L02 上下限主題 |
| Q3 | 你覺得「機器人能躺賺」是迷思還是真的？為什麼？ | 連結 L32，引導觀念辯論 |
| Q4 | 回測很漂亮、實盤翻車，你遇過嗎？你覺得是哪裡出錯？ | 連結 L27 過擬合 |
| Q5 | 你能接受帳面最大回撤幾 %？50% 你還睡得著嗎？ | 連結 L38，談風險承受度 |
| Q6 | 你最想我下一支拆哪個策略？網格／定投／套利／再平衡？ | 蒐集選題、提高回訪 |

### B. 投票題（社群分頁 Poll，2~4 選項）

| 編號 | 題目 | 選項 |
|---|---|---|
| P1 | 你現在主力跑哪種策略？ | 網格／定投／套利中性／都還在觀望 |
| P2 | 你判斷現在行情偏向？ | 震盪／趨勢／看不懂正在學 |
| P3 | 開機器人你最怕哪一個？ | 被套牢／爆倉／手續費吃光／看不懂參數 |
| P4 | 學量化你卡在哪？ | 不會設參數／不會回測／不懂風控／沒程式底子 |
| P5 | 你比較想看哪種片？ | 手把手操作／策略拆解／回測實測／純觀念 |
| P6 | 你能接受的最大回撤？ | <10%／10–30%／30–50%／無所謂長期就好 |

### C. 互動小活動（合規、不抽現金不誘導買量）

- **「設定健檢」徵集**：請觀眾留言自己的網格參數（區間、格數、標的），我挑幾個在下支影片用教學角度公開健檢（事先聲明：為觀念示範非個人化投顧）。
- **「迷思破解」票選**：列出 5 個常見迷思，讓觀眾投票最想破解哪一個，得票最高的做成 Shorts。
- **留言精選回覆**：每支影片置頂一則最有料的觀眾留言，鼓勵高品質討論而非洗版。

## 四、發文時機建議（台灣時間 UTC+8）

### 每日節奏（搭配「每日 1 長 + 3~5 短」）

| 時段 | 動作 | 理由 |
|---|---|---|
| 07:30–08:30 | 發 1 支 Shorts ＋ Threads 短文 | 通勤滑手機高峰，短內容易被吃 |
| 12:00–13:00 | 發 1 支 Shorts | 午休空檔流量第二高峰 |
| 19:00–21:00 | 發長片 ＋ 社群分頁貼文（含投票） | 晚間黃金時段，長片完播率與互動最佳 |
| 21:00–22:30 | IG／FB 導流＋回覆當日留言 | 晚間互動高峰，回留言衝演算法 |
| 22:30 前 | 補 1~2 支 Shorts（視排程） | 拉滿每日發片量、餵 Shorts 演算法 |

### 每週重點

- **週一**：發「本週主題預告 + 投票」（P5 想看哪種片），用投票決定本週主打。
- **週三**：發「觀念辯論型」社群貼文（Q3／Q4），週中互動通常較低，用爭議觀念拉留言。
- **週五–週六晚**：主打「手把手操作／策略拆解」長片，週末停留時間長、完播率高。
- **週日**：發「設定健檢」徵集或迷思票選，蒐集下週素材並維持社群熱度。

### 平台差異微調

- **Threads**：一天可多則但分散時段（早、午、晚各 1），口語拋問，忌連發洗版。
- **IG**：圖卡／Reels 晚間 19:00–22:00 最佳，重收藏與 hashtag，貼文頻率每日 1 則為宜。
- **FB**：社團發文避開深夜，挑午間或晚間，且遵守社團規則、不重複轟炸多個社團。
- **Dcard**：投資版晚間 20:00–24:00 活躍，以原創討論為主，一個主題一篇，勿短期重複發類似文。

### 通則

- 新影片上架後 **前 60 分鐘**主動回覆每一則留言，互動訊號對演算法最關鍵。
- 跨平台導流**錯開時間**而非同分同秒全平台轟炸，降低被判定 spam 風險。
- 任何聯盟連結貼文都要帶風險聲明與「合作連結」標註，無一例外。
