# 02｜SEO 與演算法策略

> 頻道：量化阿森｜Carson Quant
> 語言：繁體中文
> 主題：量化交易 / 自動交易（網格、定投、派網 Pionex、回測、風控）
> 誠信鐵則：不編造個人實測損益、不保證收益、不喊單；聯盟連結一律軟推並附風險聲明；以教學與觀念為主，引用須標來源。

## 一、頻道核心關鍵字群與長尾詞

### 1.1 核心關鍵字群（Pillar，撐起頻道主題權重）

| 群組 | 核心詞 | 搜尋意圖 |
|---|---|---|
| 自動交易 | 自動交易、量化交易、交易機器人、自動下單 | 想找「不用盯盤」的工具與觀念 |
| 網格交易 | 網格交易、網格機器人、grid trading、震盪行情 | 想懂網格邏輯與設定 |
| 定投 | 定期定額、DCA、定投策略、加密貨幣定投 | 想用紀律累積部位 |
| 派網 Pionex | 派網、Pionex、派網教學、Pionex 機器人 | 平台操作與評價 |
| 回測 | 回測、backtesting、策略驗證、歷史數據 | 想驗證策略是否可行 |
| 風控 | 風險管理、資金控管、停損、爆倉、部位大小 | 想避免虧光的觀念 |

> 原則：每支影片鎖定「1 個核心詞 + 2~3 個長尾詞」，避免一支影片塞滿所有關鍵字稀釋主題。

### 1.2 長尾詞庫（依意圖分類）

**入門 / 觀念（What / Why）**
- 網格交易是什麼、網格交易原理、網格交易適合什麼行情
- 定期定額是什麼、定投跟一次買進差別
- 量化交易新手入門、交易機器人安全嗎、自動交易會被割韭菜嗎
- 派網是什麼、Pionex 是哪國的、派網安全嗎

**操作 / 教學（How）**
- 派網網格機器人設定教學、Pionex 網格參數怎麼設
- 網格上下限怎麼抓、網格格數怎麼設、網格區間設定
- 定投機器人設定、派網定投教學
- 回測怎麼做、TradingView 回測教學、策略回測步驟

**比較 / 選擇（Compare）**
- 網格 vs 定投、現貨網格 vs 合約網格、派網 vs 幣安網格
- 哪個交易機器人好用、免費網格機器人推薦

**問題 / 風險（Risk / Troubleshoot）**
- 網格交易缺點、網格交易會虧錢嗎、網格被套牢怎麼辦
- 合約網格爆倉、網格交易風險、自動交易風險
- 套利機器人是詐騙嗎、保證收益是真的嗎

> SEO 心法：長尾詞競爭低、轉換高，且天然符合「教學/觀念」誠信定位。標題優先用「問題型長尾詞」，因為它直接對應使用者在搜尋框打的字。

## 二、長片 vs Shorts 的標題公式與縮圖文案原則

### 2.1 長片（Long-form，8~20 分鐘）

**標題公式（擇一）**
1. 〔關鍵字〕+ 〔結果/承諾，不保證收益〕+ 〔對象〕
   - 例：「網格交易是什麼？3 分鐘搞懂震盪行情怎麼自動賺價差｜新手必看」
2. 〔數字/清單〕+ 〔關鍵字〕+ 〔痛點〕
   - 例：「網格交易 5 個新手最常踩的雷，第 3 個最容易爆倉」
3. 〔問題型〕+ 〔反直覺鉤子〕
   - 例：「派網安全嗎？拆解平台架構＋它怎麼賺你的錢」

**標題規則**
- 全長控制在 60 字元內（行動裝置只顯示前約 40 字元，重要關鍵字放最前面）。
- 嚴禁「保證」「穩賺」「無腦躺賺」「日賺 X%」等字眼（違反誠信鐵則，也易被演算法判為釣魚）。
- 數字、問號、｜分隔符可提高點擊；但全標題最多 1 個情緒鉤子，避免標題殺人。

**縮圖文案原則**
- 字數：3~6 個大字，一眼可讀（手機縮圖只有指甲大）。
- 結構：主詞 + 衝突/疑問。例：「網格＝穩賺？」「定投真的有用嗎」。
- 配色：高對比（深底亮字或亮底深字），品牌色固定一組，建立辨識度。
- 一致性：左上或右下固定放「量化阿森」小 logo / 頭像；同系列用同一視覺模板。
- 誠信：縮圖不可出現假對帳單、假損益截圖、紅色暴利數字。

### 2.2 Shorts（短片，< 60 秒，直式）

**標題公式**
- 〔超短鉤子〕+ 〔1 個核心關鍵字〕+ #標籤
  - 例：「網格交易最大的誤會 #網格交易 #量化交易」
- Shorts 標題以「停下手指」為目標，前 3 秒就是標題的延伸。

**標題規則**
- 標題更短（30~40 字元），1 個關鍵字即可。
- 描述放 1~2 句重點 + 3~5 個 hashtag。
- 開頭 1 句必須是鉤子（提問 / 反常識 / 數字），不要鋪陳。

**縮圖/封面文案原則（Shorts 多靠首幀）**
- 首幀=封面：第一幀就要有大字標題，2~4 字。
- 直式構圖：文字置中偏上，避免被底部 UI（標題列、頻道名）擋住。
- 字級放到最大，因為 Shorts 多在資訊流快速滑過。

## 四、已上線 6 支舊片 標題／標籤優化建議

> 註：以下為「優化方向模板」。實際舊片標題請以後台清單為準，套用同一套改寫邏輯：①把核心關鍵字提到最前 ②加入長尾問題詞 ③移除任何疑似保證收益字眼 ④補齊主題標籤群。

| 舊片（類型） | 常見問題 | 建議優化標題（方向） | 建議標籤補強 |
|---|---|---|---|
| 舊片 1：網格介紹 | 標題太籠統、無關鍵字前置 | 網格交易是什麼？新手必懂的自動低買高賣邏輯 | 補：grid trading, 震盪行情, 新手教學 |
| 舊片 2：派網開箱 | 含「躺賺/被動收入」誘導字 | 派網 Pionex 怎麼用？平台機制與費用一次看 | 移除誘導字；補：派網教學, 交易機器人 |
| 舊片 3：定投分享 | 像 vlog、無搜尋詞 | 定期定額怎麼開始？定投策略與常見迷思 | 補：DCA, 加密貨幣定投, 資金控管 |
| 舊片 4：回測展示 | 標題藏結果（暗示獲利） | 策略回測教學：如何驗證一個交易策略 | 移除損益暗示；補：backtesting, 過度擬合 |
| 舊片 5：機器人實測 | 標題含個人損益數字 | 交易機器人運作原理：它到底怎麼下單 | 移除損益數字（誠信）；補：自動交易, 量化交易 |
| 舊片 6：風控雜談 | 關鍵字弱、標題冗長 | 資金控管比策略更重要？自動交易的風險管理 | 補：風險管理, 停損, 部位大小 |

**舊片共通優化動作**
1. 標題：核心關鍵字置前 + 1 個長尾問題詞，刪除任何「保證/穩賺/躺賺/實測賺 X」。
2. 描述（前 2 行最關鍵）：第一句即放主關鍵字，並加 1 句風險聲明（「本影片為教學與觀念分享，非投資建議」）。
3. 標籤：補齊「主題群 + 平台 + 風險」三類，控制在 10~15 個。
4. 章節（Chapters）：替長片加時間戳，提高停留與關鍵字覆蓋。
5. 縮圖：若舊縮圖含假對帳單/暴利數字，立即換成符合誠信的版本。
6. 置頂留言：補上聯盟連結（軟推）＋風險聲明＋「非投資建議」。

## 五、每日 SEO 檢查清單

**發布前（每支影片）**
- [ ] 標題：核心關鍵字置前？含 1 個長尾問題詞？字數 ≤ 60？
- [ ] 標題：無「保證/穩賺/躺賺/喊單/個人實測損益」字眼？
- [ ] 描述前 2 行：含主關鍵字 + 一句鉤子？
- [ ] 描述：含風險聲明（非投資建議）＋聯盟連結軟推＋引用來源？
- [ ] 標籤：10~15 個，涵蓋主題群／平台／風險三類？
- [ ] 縮圖：3~6 字、高對比、品牌一致、無假損益截圖？
- [ ] 長片加章節時間戳；Shorts 首幀有大字封面？
- [ ] 字幕 / CC：上傳正確繁中字幕（利於搜尋與停留）？
- [ ] 播放清單：歸入對應系列（網格／定投／派網／回測／風控）？
- [ ] 結束畫面 / 卡片：導向同主題舊片，做內部串連？

**每日例行（頻道層級）**
- [ ] 查看昨日上片的 CTR 與平均觀看時長，CTR 偏低先測縮圖/標題 A/B。
- [ ] 掃留言區：回覆前 3 則高互動留言，挑 1 則做下支 Shorts 選題。
- [ ] 看「觀眾搜尋的關鍵字」報表，補進長尾詞庫。
- [ ] 檢查是否有舊片流量回升，趁勢更新標題/縮圖或拍延伸片。
- [ ] 確認所有含聯盟連結的影片，置頂留言/描述風險聲明仍在。

**每週覆盤**
- [ ] 找出 CTR 最高與最低各 1 支，歸納標題/縮圖規律。
- [ ] 更新關鍵字庫：新增 5~10 個長尾詞，淘汰無流量詞。
- [ ] 檢視 5 大系列的影片數量是否均衡，補足缺口。
- [ ] 抽查 3 支舊片，套用第四節優化動作。
