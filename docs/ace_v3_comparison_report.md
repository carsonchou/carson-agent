## V3 問題分析

### ❌ V3 整體退步的核心原因

1. **MTF 4H 過濾反效果**  
   - V3 新增的 4H MTF 過濾在日線策略上造成 **訊號延遲 + 假突破增加**
   - ETH 從 922% 暴跌至 34%，主因是 MTF 過濾砍掉了大部分有效多頭訊號

2. **RSI 雙重確認過於嚴苛**  
   - RSI≥48 做多門檻在盤整行情中大量錯過初期反彈
   - LINK/UNI/ATOM 等幣種從盈轉虧

3. **MA100 比 MA200 更容易穿越**  
   - MA100 對噪音更敏感，在震盪行情中頻繁切換方向 → 假訊號增加

4. **confirmBar=1 並未改善**  
   - V3 的 77~80 筆交易仍遠多於 V2 的 28 筆，但勝率更低
   - 說明多重過濾條件彼此抵消，反而增加了低品質交易

### ✅ V3 唯一明顯優勢：AVAX

- AVAX V3 +351.90% 大幅優於 V2 -78.78% 和 V1 +103.47%
- 原因：AVAX 的 4H 趨勢過濾效果特別好，有效過濾了 2022-2023 的假多頭

## 結論與建議

### V2 仍是整體最優策略
V2 在 8/14 幣種上獲勝，V3 只在 AVAX 一幣勝出。V3 的多重過濾機制在日線策略上弊大於利。

### V4 建議方向

1. **移除或選擇性啟用 MTF 過濾**  
   - 只對 AVAX/BTC/NEAR 類趨勢型幣種開啟 4H MTF
   - 對 ETH/XRP/ADA 關閉 MTF

2. **放寬 RSI 確認**  
   - 將 RSI 門檻改為：做多 RSI≥40，做空 RSI≤60（更寬鬆）

3. **回復 MA200 + 補充 MA100 雙軌判斷**  
   - 主趨勢用 MA200，動能確認用 MA100

4. **降低時間框架至 4H**  
   - 日線訊號太少（V2 平均 16 筆），4H 訊號更多且停損更精準
   - 預期勝率可從 35% 提升至 40%+

5. **分幣種部署**  
   - ETH/XRP/BTC/DOGE/LTC/UNI：用 V2 參數
   - AVAX/BTC：用 V3 或加 MTF 過濾的特製版
