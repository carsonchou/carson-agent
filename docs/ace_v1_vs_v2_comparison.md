## v2 優化內容
1. **入場閾值提高**：±40 → ±50（只進高品質訊號）
2. **MA200 順勢過濾**：只做多在 MA200 之上，只做空在 MA200 之下
3. **量能確認**：成交量需 > 20日均量才進場
4. **score=0 出場**：分數回到 0 立即出場（不等對向訊號）
5. **ATR 停損**：2×ATR 保護（限制單筆虧損）
6. **ATR 停利**：4×ATR 目標（保住利潤）
7. **移動停損**：1.5×ATR 移動停損（讓贏家跑）

## 分析：v2 哪裡更好，哪裡更差

### ✅ v2 明顯改善（7 幣）
- **BTC**：MA200 過濾掉 2022 熊市空單假訊號，回撤從 60% 砍到 30%
- **ETH**：MA200 + 停損讓勝率從 41% → 64%，回撤從 74% 砍到 29%
- **XRP / LTC / UNI**：從虧損變盈利，停損機制大幅保住資金
- **SOL / DOGE / ATOM**：仍虧，但虧得少很多（停損在發揮作用）

### ❌ v2 表現較差（3~4 幣）
- **ADA / AVAX / LINK / DOT**：問題在 **MA200 過濾過早** ——這幾個幣在 2022 下跌期間曾短暫突破 MA200，ACE v1 抓到了早期多頭訊號，v2 的 MA200 過濾直接砍掉這些訊號

### 核心結論
> v2 大幅改善「過度交易」問題：從 v1 的 28-47 筆縮減到 v2 的 5-16 筆，**品質 >> 數量**  
> 平均最大回撤：v2 從 ~84% 降低到 ~50%（重大改善）  
> 勝率中位數：v1 ~34% → v2 ~37%，提升有限但高峰更高（ETH 64%、UNI 60%）

## 建議：如何進一步提升勝率

### 短期可行調整
1. **對 ADA/AVAX/LINK/DOT 關閉 MA200 過濾**（或改用 MA100）
2. **提高停利至 5×ATR**（讓 ETH/XRP 類大趨勢幣的獲利更豐厚）
3. **增加 RSI 確認**：做多時 RSI > 45，做空時 RSI < 55

### 根本性改進（v3 方向）
- **加入時間過濾**：只在特定月份（如 1-4 月、10-12 月）交易（加密市場季節性）
- **切換模式**：大多頭只做多，大空頭只做空（用 Monthly 趨勢判斷）
- **降低時間框架**：改為 4H 圖，訊號更多、止損更精確、勝率理論上更高
