## 關鍵觀察

### 優勢
- ETH、ADA 在牛市趨勢明顯的行情中表現極佳
- BNB 以極低勝率（23.8%）仍達到正收益，代表「大賺小虧」的特性發揮
- POL 樣本量少（8筆），但勝率 37.5% + 獲利因子 1.27 表現穩健

### 弱點
- **整體勝率偏低（平均 ~34%）**：ACE 策略本質是趨勢跟蹤，非高頻高勝率策略
- **最大回撤普遍極高（~80%）**：日線策略在加密貨幣高波動環境中，不加倉位控制的話回撤非常大
- **LTC、SOL、DOGE、ATOM 明顯虧損**：可能因這些幣種的波動噪音較大，使得 ACE 的複合分數頻繁假突破

### 改進建議
1. **加入停損機制**：如 ATR×2 停損，可大幅降低最大回撤
2. **只交易高品質訊號**：提高 longTh/shortTh 閾值（如 50/-50）
3. **選擇性幣種**：聚焦 ETH、ADA、AVAX 等表現好的幣種
4. **縮短時間框架**：4H 或 1H 可能提供更多訊號並減少單筆虧損幅度
