## 一、這是什麼

一支基於**三重 SuperTrend（1:2.5:6 ATR 週期比）** 的趨勢跟蹤**策略（strategy）**，核心理念：用多尺度趨勢結構過濾方向、波動率目標化控制風險、並**讓獲利奔跑**換取正期望值。

跟市面上常見三指標策略最大的不同：**這支的每個版本都實際上 TradingView 跑過真實回測驗證**，不是靠紙上推導或 AI 自我評分。

## 三、核心機制（從強到弱）

1. **三重 SuperTrend 方向結構**：ST1(快/noise) / ST2(swing) / ST3(regime)，週期比 1:2.5:6。
2. **Regime 進場過濾**：Efficiency Ratio (ER) + ADX（含遲滯）+ ST3 三票多數決 + DI 方向閘——只在「真趨勢」進場。
3. **波動率目標化倉位**：以年化波動率標準化部位、maxGross 上限、高波動剎車——把每筆風險拉齊。
4. **讓獲利奔跑的出場（這是 edge 主來源）**：
   - 部分止盈只減 20%（TP1 @1.5R 減 10%、TP2 @4.5R 減 10%），保留 80% 核心倉吃右尾。
   - Chandelier 移動停損 + R 階梯收緊（>2R 才較緊，2.5×ATR 呼吸空間）。
   - Peak-R 回吐鎖（~1.5R 才啟動，低 R 不早鎖）。
5. **快砍虧損**：初始停損緊（3×ATR）、達 1R 即保本——確保輸的是小虧。
6. **time-stop + 出場冷卻**：死單 45 根 K 內 R<0.5 退場、出場後冷卻 4 根避免追單。

## 四、怎麼用（TradingView）

1. 開 TradingView 圖表 → 右側 **Pine Editor**。
2. 把 `triple_supertrend_v4_FINAL.pine` 全文貼進去 → **Add to chart**。
3. 開 **Strategy Tester** 看回測績效；圖上會有**狀態儀表板**即時顯示 regime/倉位/R/出場狀態。
4. 輸入參數已分組（趨勢結構 / Regime 濾網 / 倉位 / 出場 / 顯示），每個 input 有 tooltip 說明。

### 建議設定
- **商品/週期**：加密貨幣大幣 **4H**（BTC/ETH 已驗證）。其他週期/商品請先自行回測。
- **手續費/滑價**：已內建 `commission 0.05% + slippage 2`，貼近現實；換交易所請依實際調整。
- **初始資金/風險**：依自己帳戶調 `風險%` 與 `maxGross`，別一上來就滿槓桿。

## 五、誠實的定位與風險

- **這是「穩健但溫和」的正期望趨勢策略**：PF 1.15~1.31 代表每虧 1 元賺回 1.15~1.31 元——會賺、抗過擬合、可實盤，但**不是碾壓級暴利**。真實市場本來就難有暴利而穩健的東西。
- **趨勢策略的本性**：勝率約 48%（輸的單比贏的多），靠**少數大趨勢的大獲利**賺錢。會經歷連續小虧的盤整期，要扛得住。
- **回撤**：BTC 7% / ETH 11.6%——可接受但非零，務必用能承受的部位。
- **未驗證的就是未驗證**：其他商品/週期、未來行情、實盤滑價都需自己再確認。本策略不保證未來績效。
